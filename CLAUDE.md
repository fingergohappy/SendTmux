1. 插件名称

SendToTmux（可根据实际命名修改）

2. 插件目标

允许用户在 VS Code 中选中任意文本，通过快捷键将选中的内容发送到指定的 Tmux 会话/窗口/面板中执行。

3. 功能描述

发送选中内容

用户在编辑器中选中一段文本。

使用快捷键将选中的内容发送到指定的 Tmux session/window/panel。

配置 Tmux 目标

用户可以在插件设置中指定 Tmux：

会话名（session）

窗口编号或名称（window）

面板编号（panel）

支持全局配置和工作区单独配置。

快捷键触发

用户可自定义快捷键。

快捷键触发后，将当前选中的文本发送到 Tmux。

多行文本支持

支持一次发送多行选中内容。

可配置发送方式（逐行发送或一次性发送）。

错误提示和验证

未安装 Tmux 时提示安装。

session/window/panel 不存在时给出错误提示。

可选择执行前确认目标 session/window/panel。

4. 用户体验细节

插件可根据需要提示用户确认目标 panel（一次性或每次）。

支持快捷键和命令面板双触发方式。

可记录最近使用的 session/window/panel，方便快速选择。

提供简单易用的配置界面，无需修改插件代码。
